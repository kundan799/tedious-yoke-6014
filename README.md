## Tech stack :
### Frontend :
- React
- CSS-Module
- Chakra-UI
- Axios
- react-router-dom





## Run Locally :
Clone the Project
```
https://github.com/kundan799/tedious-yoke-6014
``` 

Install Dependencies
```
npm install
```
Run App
```
npm Start
```



Now go to server Directory and run the server
```
npm run server
```
